# LeetCode SQL Solutions

A curated collection of SQL solutions to LeetCode problems, written with a focus on:

🔍 Clear logic and readability <br>
⚡ Efficient query patterns<br>
🧠 Window functions & advanced SQL techniques<br>
📊 Real interview-ready approaches<br>


**🚀 Why this repository?**<br>

This repo is not just about solving problems — it focuses on:<br>

Understanding how SQL thinks (set-based logic)<br>
Mastering window functions (ROW_NUMBER, LAG, LEAD)<br>
Learning pattern-based solutions for interviews<br>
Building intuition for real-world data queries<br>

🔹 **Joins** <br>

🧩 **Problem-Solving Patterns**<br>

Instead of memorizing queries, this repo focuses on patterns like:<br>

🔸 **Consecutive Numbers / Streaks** <br>
Convert consecutive rows into groups using:<br>

id - ROW_NUMBER() OVER (ORDER BY id)<br>
🔸 **Self Join Pattern**<br>

Used for:<br>

Comparing adjacent rows<br>
Detecting patterns in sequences<br>
🔸 **Window Function Pattern**<br>

Used for:<br>

Ranking<br>
Running totals<br>
Moving comparisons<br>
